2. 데이터 저장 및 보관

  • 저장소: Cloudflare D1 (SQLite 호환) — 한국 또는 아시아 지역 데이터센터에 분산 저장
  • 전송: 모든 클라이언트↔서버 통신은 HTTPS/WSS로 암호화 (Cloudflare TLS)
  • 백업: 매일 03:00 KST에 자동 백업, 최대 30일 보관
  • 보관 기간: 사용자가 삭제를 요청하기 전까지, 또는 시즌 종료 후 통계 목적으로 무기한 보관 (개인 식별 정보는 익명화 가능)

3. 제3자 공유

서비스는 사용자의 개인정보를 제3자에게 판매·임대·공유하지 않습니다. 다만 다음 외부 서비스를 사용하는 과정에서 일부 정보가 처리됩니다:

  • Discord: 서비스 진입·인증을 위해 Discord OAuth2 및 Bot API 사용. Discord 개인정보 보호 정책
  • Riot Games: 사용자가 직접 등록한 Riot Game Name#Tag로 Riot Games API 호출. Riot Games Privacy Notice
  • Cloudflare: DNS/프록시·D1 데이터베이스. Cloudflare Privacy Policy

4. 사용자의 권리

  • 열람: /내정보 명령으로 자신에 대해 저장된 주요 정보 확인
  • 수정: /등록 명령으로 Riot 계정 매핑 갱신
  • 삭제: 운영자에게 요청 시 본인 관련 데이터를 삭제
  • 철회: Discord에서 monkey의 OAuth2 권한을 해제하면 추가 데이터 수집 중단

5. 쿠키 및 추적

서비스는 인증 유지 목적의 세션 쿠키 1개만 사용합니다. 추적용 쿠키, 광고 쿠키, 분석 쿠키(Google Analytics 등)는 사용하지 않습니다.

6. 보안

  • 모든 통신은 TLS 1.2 이상으로 암호화
  • 세션 쿠키는 HttpOnly, Secure, SameSite=None 설정으로 XSS/CSRF 방어
  • Cloudflare API 토큰·Riot API 키는 서버 환경변수로만 관리, 클라이언트에 노출 안 됨

7. 미성년자

서비스는 Discord의 13세 이상 이용 정책을 따릅니다. 13세 미만의 이용은 권장하지 않습니다.

8. 정책 변경

본 정책은 사전 고지 없이 변경될 수 있으며, 변경 시 본 페이지의 "최종 갱신" 날짜를 갱신합니다.

9. 문의

개인정보 처리에 관한 문의는 GitHub 저장소 또는 Discord 서버 관리자를 통해 주십시오.
