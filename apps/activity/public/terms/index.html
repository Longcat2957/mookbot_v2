monkey 이용약관

최종 갱신: 2026-04-30

1. 서비스 개요

monkey(이하 "서비스")은 Discord 서버 내에서 League of Legends 내전(Bo3, N v N)의 모집·진행·결과 기록·MMR 관리 기능을 제공하는 비공식 봇 및 Discord Activity입니다. Riot Games의 공식 서비스가 아니며 Riot Games와 무관합니다.

2. 이용 자격

서비스는 Discord 약관을 준수하는 모든 사용자가 자유롭게 이용할 수 있습니다. 서비스 이용 자체로 별도의 가입 절차는 없으며, Discord 계정과 (선택적으로) Riot 계정을 연결하는 것으로 등록됩니다.

3. 사용자의 의무

  • 다른 사용자의 정보를 도용하거나 허위 등록하지 않을 것
  • 봇/서비스의 정상 동작을 방해하는 행위(스팸, 어뷰징, 자동화 공격)를 하지 않을 것
  • 운영자가 정한 서버별 내전 규칙을 따를 것

4. 데이터 보관 및 삭제

서비스는 내전 운영에 필요한 최소한의 데이터(Discord ID, 표시 이름, Riot Game Name#Tag, 내전 참여 기록, MMR)를 저장합니다. 자세한 내용은 개인정보 보호 정책을 참조하십시오.

사용자는 운영자에게 요청하여 자신의 데이터 삭제를 받을 수 있습니다.

5. 책임의 한계

서비스는 "있는 그대로(as-is)" 제공되며, 다음 사항에 대해 책임지지 않습니다:

  • 서비스 중단, 데이터 손실, MMR 산정의 오류
  • Riot Games API의 가용성 및 정확성
  • Discord API/Activity 플랫폼의 변경에 따른 기능 손실
  • 사용자 간의 분쟁 (내전 결과, 매너 이슈 등)

6. 약관 변경

본 약관은 사전 고지 없이 변경될 수 있으며, 변경 후 서비스를 계속 이용하는 경우 변경된 약관에 동의한 것으로 간주됩니다. 중대한 변경 사항은 봇의 공지사항을 통해 안내합니다.

7. 준거법

본 약관에 관한 분쟁은 대한민국 법령을 준거법으로 합니다.

8. 문의

문의는 GitHub 저장소 또는 Discord 서버 관리자를 통해 주십시오.
